## Como Acessar o Site

O servidor HTTP local já está rodando na porta 8000. Para acessar o site:

1. **Via navegador web:**
   - Abra seu navegador
   - Acesse: `http://localhost:8000` ou `http://127.0.0.1:8000`

2. **Via linha de comando (teste):**
   ```bash
   curl http://127.0.0.1:8000
   ```
